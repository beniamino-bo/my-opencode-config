# Operational Rules

- **Deletion safeguard:** Before deleting any files or directories, you must
  first ask the user for explicit permission. After receiving their
  confirmation, you must ask a second time ("Are you sure?") before proceeding with the deletion.

  **Avoid bypassing permissions:** If you do not have permission to execute some commands 
  or actions, do not try to work around the restriction with similar commands, different tools,
  or other methods that achieve the same effect. Always respect the configured permissions.
